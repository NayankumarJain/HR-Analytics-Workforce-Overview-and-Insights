## Project Objective
The objective of this project is to analyze workforce composition and hiring trends using an interactive Power BI dashboard.

## Tools Used
- Power BI
- Excel / CSV
- DAX

## Dashboard Pages

### HR Overview
- Total Employees: 22K
- Remote vs Headquarters workforce distribution
- Average employee age
- Hiring trends over the years
- Gender diversity

### Workforce Insights
- Employee distribution by department
- Employee distribution by state
- Employee distribution by race

## Key Insights
- Majority of employees are based at headquarters
- Hiring trends show steady growth over the years
- Engineering is the largest department
- Workforce diversity is visible across gender, race, and geography

## Conclusion
This project demonstrates how HR data can be transformed into meaningful insights using Power BI dashboards to support workforce analysis.
